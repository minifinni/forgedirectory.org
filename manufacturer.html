Shapeways

⭐⭐⭐⭐⭐ API Maturity 5/5

The leading 3D printing service with 60+ materials. Best for detailed figurines, full-color sandstone prints, jewelry, and consumer products.

About

Shapeways is the leading on-demand 3D printing service, founded in 2007 and headquartered in New York City. With over 60 materials ranging from standard plastics to precious metals, Shapeways serves creators, businesses, and enterprises worldwide.

The Shapeways API enables developers to integrate 3D printing capabilities directly into their applications. Whether you're building a custom product configurator, a print-on-demand store, or an automated prototyping pipeline, the Shapeways API provides full access to their manufacturing network.

Particularly well-suited for detailed figurines, jewelry, miniatures, and full-color consumer products thanks to their extensive material library and high-quality production standards.

API Capabilities

  • ✓ 3D model upload and validation (STL, OBJ, 3MF, DAE, X3D)
  • ✓ Instant quoting based on material, volume, and geometry
  • ✓ Order placement and lifecycle management
  • ✓ Material and finish selection programmatically
  • ✓ Shipping management and tracking
  • ✓ Webhook support for status updates
  • ✓ White-label shipping options
  • ✓ OAuth 2.0 authentication

Quick Start Example

JavaScript 
// Initialize Shapeways client
const shapeways = require('shapeways-api');

const client = new shapeways.Client({
  consumerKey: 'YOUR_CONSUMER_KEY',
  consumerSecret: 'YOUR_CONSUMER_SECRET',
  oauthToken: 'YOUR_OAUTH_TOKEN',
  oauthSecret: 'YOUR_OAUTH_SECRET'
});

// Upload a model
const model = await client.uploadModel({
  file: './model.stl',
  title: 'Custom Figurine',
  description: 'Wedding cake topper'
});

// Get instant quote
const quote = await client.getQuote({
  modelId: model.modelId,
  materialId: '6c38f9d4-5a1e-4e2b-8c3d-9f4e5d6c7b8a' // Full Color Sandstone
});

console.log(`Price: $${quote.price}`);
console.log(`Lead time: ${quote.leadTime} days`);
